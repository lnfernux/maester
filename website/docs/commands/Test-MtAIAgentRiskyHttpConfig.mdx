## SYNOPSIS

Tests if AI agents have risky HTTP configurations.

## SYNTAX

```powershell
Test-MtAIAgentRiskyHttpConfig [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Checks all Copilot Studio agents for HTTP actions that connect to non-standard
ports or non-connector endpoints.
HTTP actions to unexpected destinations may
indicate data exfiltration, command-and-control communication, or misconfigured
integrations.

### EXAMPLE 1
