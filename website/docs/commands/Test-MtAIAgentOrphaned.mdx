## SYNOPSIS

Tests if AI agents are orphaned.

## SYNTAX

```powershell
Test-MtAIAgentOrphaned [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Checks all Copilot Studio agents for those whose owner accounts no longer
exist or are disabled in Entra ID.
Orphaned agents lack active governance
and may drift from security policies without anyone responsible for
maintaining them.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtAIAgentOrphaned
```
